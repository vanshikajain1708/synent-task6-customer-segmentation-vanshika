## Task 6: Customer Segmentation

### Problem Statement
Group customers based on their Annual Income and Spending Score using clustering.

### Dataset
Mall Customers Dataset

### Approach
- Data Preprocessing & Scaling
- Elbow Method to find optimal number of clusters
- K-Means Clustering (5 clusters)
- Cluster Visualization

### Key Insights
- 5 distinct customer segments found
- High Income + High Spending = Premium Customers
- High Income + Low Spending = Potential Customers

### Technologies Used
Python, Scikit-learn, Pandas, Seaborn, Matplotlib
